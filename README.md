# kMenu

KMenu For Redzone/Combat FiveM Servers. Runs at 0.00


## Installation
1) Add the folder to your server resources.
2) Add `ensure kmenu` to your server.cfg

### Dependencies
1) MenuV - https://github.com/ThymonA/menuv
